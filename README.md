
### Superhero Dueler
#### ACS 1111

Built with Python to demonstrate OOP
#### How to run


Setup:
``` bash
git clone https://github.com/dejmedus/acs-1111-superhero-dueler.git
cd acs-1111-superhero-dueler
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
```

App:
``` bash
python3 area.py
```

Tests:
``` bash
pytest
```
